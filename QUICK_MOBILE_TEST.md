# تست سریع روی موبایل 🚀

## روش سریع (3 مرحله)

### 1️⃣ اجرای سرور
```bash
npm run dev:mobile
```

یا در ویندوز:
```powershell
$env:HOSTNAME="0.0.0.0"; npm run dev
```

### 2️⃣ پیدا کردن IP آدرس

**ویندوز:**
```cmd
ipconfig
```
به دنبال `IPv4 Address` بگردید (مثال: `192.168.1.100`)

**مک/لینوکس:**
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

### 3️⃣ باز کردن در موبایل

در مرورگر موبایل (Chrome/Safari) وارد شوید:
```
http://192.168.1.100:3000
```
(IP آدرس خود را جایگزین کنید)

---

## نکات مهم ⚠️

✅ موبایل و کامپیوتر باید به **یک Wi-Fi** متصل باشند  
✅ اگر کار نکرد، Firewall را چک کنید  
✅ IP آدرس را از console سرور هم می‌توانید ببینید
